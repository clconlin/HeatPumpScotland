# Heat Pump Installation Scotland

A static website for a Scottish heat pump installation company.

## Pages
- **Home** — Hero, services overview, trust signals
- **Services** — Air source, ground source, servicing, grants
- **About** — Company story, accreditations, values
- **Testimonials** — Customer reviews
- **Gallery** — Instagram feed embed (auto-updating)
- **Contact** — Quote request form via Formspree

## Setup
1. Open in VS Code
2. Open `index.html`
3. In the terminal run `start index.html`
